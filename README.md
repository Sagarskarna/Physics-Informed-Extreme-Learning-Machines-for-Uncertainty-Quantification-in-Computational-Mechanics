# Physics-Informed Extreme Learning Machines (PIELM) for Uncertainty Quantification in Computational Mechanics

This repository contains the implementation, numerical experiments, and presentation material for my thesis work on **Physics-Informed Extreme Learning Machines (PIELM)** applied to **Uncertainty Quantification (UQ)** in computational mechanics.

The work investigates the capability of PIELM to efficiently solve governing differential equations while simultaneously capturing uncertainty in physical systems.

---

## 📌 Thesis Objective

The main objective of this work is to explore whether **PIELM can be extended for efficient uncertainty quantification**, while maintaining:

* Fast computation
* Physical consistency
* Accurate statistical prediction

---

## ⚙️ Topics Covered

* Extreme Learning Machine (ELM)
* Physics-Informed ELM (PIELM)
* Constrained optimization using KKT conditions
* Deterministic PDE solving
* Uncertainty Quantification (UQ)
* Sensitivity Analysis
* Monte Carlo Simulation
* Latin Hypercube Sampling (LHS)
* ECDF and statistical validation

## 🧠 Key Features

* Physics-informed neural approximation
* No backpropagation or iterative optimization
* Fast linear-system-based solution
* Exact enforcement of boundary conditions using KKT formulation
* Efficient uncertainty propagation
